== Description ==

Filosofo Comments Preview lets you preview WordPress comments before you submit them.

== Installation ==

1. Upload `filosofo-comments-preview.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Navigate to Options > Comments Preview for additional options.
